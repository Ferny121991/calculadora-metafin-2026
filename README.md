# Calculadora Financiera "MetaFin 2026 Pro"

Una calculadora financiera web desarrollada en React, diseñada para proyectar el pago de deudas y organizar tus finanzas personales aplicando metodologías como la "Bola de Nieve" y priorizando ingresos adicionales (Side Hustles).

## ✨ Características Principales

*   **Persistencia de Datos Automática:** Todos tus progresos, como pagos de quincena, rachas, ingresos y gastos, se guardan internamente en tu navegador (`localStorage`), evitando la pérdida de información si cierras la pestaña.
*   **Gestión de Ingresos Extras (Side Hustle Tracker):** Agrega ingresos no rutinarios (como viajes de Uber, ventas casuales, regalos) que potencian tu "Poder de Destrucción Mensual" contra las deudas.
*   **Temas Visuales Dinámicos:** La interfaz cambia de color sutilmente basándose en tu progreso. Se vuelve progresivamente "Esmeralda" a medida que reduces tu deuda e incrementas tus ingresos adicionales para motivarte visual y psicológicamente.
*   **Sistema de Logros:** Desbloquea medallas y lanza confeti completando meses perfectos y alcanzando hitos monetarios con ingresos extra.
*   **Previsiones "Bola de Nieve":** Proyecciones en tiempo real que te ilustran cuán rápido podrás pagar préstamos específicos dependiendo de cuánto puedas aportar mensualmente.

## 🛠 Tecnologías Utilizadas

*   **React:** Para la interfaz de usuario interactiva y el manejo de estado (`useState`, `useEffect`).
*   **Tailwind CSS:** Para un diseño moderno, estéticamente agradable y totalmente responsivo (adaptable a móviles).
*   **Lucide Icons:** Para la iconografía del sitio.
*   **Canvas Confetti:** Para efectos de celebración motivacionales.
*   **Firebase Hosting:** Para un despliegue rápido y seguro en la web.
